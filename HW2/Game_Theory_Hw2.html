to go ifelse sequential? = false ;; check if play is simultaneous or sequential [ ask players [shoot] ;; if play is simultaneous, players shoot sequentially (random order) check-die ;; but die simultaneously ][ ;; if play is sequential, players shoot and die sequentially (specified order) ask turtle-set player 0 [shoot] ;; turtle-set command used here so asking dead players to shoot doesnt stop the simulation check-die ask turtle-set player 1 [shoot] check-die ask turtle-set player 2 [shoot] check-die ] end

to shoot ;; routine for players to pick a target (rationally) and shoot if ammo > 0 ;; only shoot if player has ammo [ if random 100 < (accuracy * 100) ;; players have same accuracy regardless of target, so check if shot hits first [ ifelse who = 0 ;; Player 1 has special shoot command because they are aware of Player 2s ammunition [ask turtle-set max-one-of other players with [ammo > 0] [accuracy] ;; Player 1 targets other player with highest accuracy, as long as they have bullets [set alive? false]][ask turtle-set max-one-of other players [accuracy] ;; Players 2 and 3 target other player with the highest accuracy [set alive? false]] ] set ammo ammo - 1 ;; if shot was successful, lose a bullet ] end

to check-die ;; routine to kill players that have been shot and to monitor Player 1s payoff if count players <= 1 [stop] ;; stop the simulation if one or no players remain ask players with [alive? = false] [ die ] ;; kill players that have been shot if player 0 != nobody [set payoff payoff + 1] ;; if Player 1 is alive, increase their payoff tick end
